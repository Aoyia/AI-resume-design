# 设置默认简历模板设计说明书

本项目旨在将用户下载目录中最新的简历数据 JSON 文件设置为系统默认简历模板。

## 变更背景

用户在本地下载目录中有一份最新的简历 JSON 数据文件：`前端(AI 应用)_4年低代码经验_杨忠源_15779736785 (1).json`。需要将其内容作为系统的默认模板加载，使得新用户打开页面或重置时，默认呈现该份简历的数据。

## 设计方案

1. **源文件路径**：
   - `/Users/neoyuan/Downloads/前端(AI 应用)_4年低代码经验_杨忠源_15779736785 (1).json`

2. **目标文件路径**：
   - `src/data/defaultResume.json`（对应绝对路径：`/Users/neoyuan/Desktop/aoyi/AI-resume-design/src/data/defaultResume.json`）

3. **操作细节**：
   - 读取源文件，将其完整内容（包含 `avatar` 的 Base64 大文件数据）直接覆盖写入目标文件。
   - 保持标准的 JSON 格式，缩进为 2 空格。

## 验证计划

- **静态验证**：检查 `src/data/defaultResume.json` 是否存在，且格式正确无语法错误。
- **动态验证**：启动本地开发服务器，打开简历应用，确认默认加载的简历内容是否已更新为最新的“杨忠源 - 4年低代码经验（前端AI应用）”简历，且头像和各个模块能正常渲染。
