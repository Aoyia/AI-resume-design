技术栈：React + Python + LlamaIndex + Elasticsearch + Milvus + PDFPlumber + Docker

核心贡献：

  • 高精度解析与切片：自研解析器攻克了复杂大文档的双栏排版与跨页表格还原难题；设计**语义感知切片（Semantic Chunking）**算法，大模型上下文召回率（Recall）提升 35%。
  • 混合检索与重排：构建基于 Elasticsearch (文本) 与 Milvus (向量) 的混合检索（Sparse + Dense Search）机制，配合 Cohere Rerank 模型，知识库问答准确度从 65% 大幅提升至 92%。
  • 极速渲染优化：面对 10 万行级向量切片日志与调试面板，采用 Web Worker 离线计算与 IndexedDB 缓存，结合前端虚拟列表策略，日志面板帧率稳定在 55fps 以上。
